Seoul Bus Route Open API

XML data from an open API about Seoul Bus Transportation.

Using python I extracted out data from the open API in XML format and parsed the data for each bus' congestion and arrival times. Processing this data, I was able to figure out which bus stations at what time of day had the most congestion.

  • Date: February 2020
  • Main Program: Python
  • Category: Open API
